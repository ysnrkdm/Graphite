# Graphite

An open source Othello application run on Command line console.

## Run on

- Linux (Ubuntu)
- macOS

## Install

```shell
git clone <this repository>
cd Graphite
make
```

## How to play

```shell
make run
```

### Commands to play

```shell
init
go
<hand, e.g., D3, e5, ps, etc...>
```

Have fun!
